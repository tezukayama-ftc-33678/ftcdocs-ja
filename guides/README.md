# Guides ディレクトリ

このディレクトリには、翻訳プロジェクトの重要なガイドドキュメントが含まれています。

---

## 📚 含まれるガイド

### `GLOSSARY.md` - 用語集

FTC用語の日本語訳と統一表記のリスト。翻訳時に参照してください。

**主な内容**:
- FTC技術用語の日本語訳
- 統一すべき表記
- 英語のまま残すべき用語

### `LICENSE_AND_LOGO_GUIDE.md` - ライセンスとロゴ使用ガイド

翻訳版のライセンスとFIRSTロゴの使用に関する重要な情報。

**主な内容**:
- 翻訳版の法的位置づけ
- FIRSTロゴの使用可否
- 免責事項の記載方法
- 権利表記のガイドライン

### `LOCAL_LLM_SETUP.md` - ローカルLLM環境のセットアップ

Ollamaを使用したローカルLLM環境の構築手順。

**主な内容**:
- Ollamaのインストール方法
- LLMモデルのダウンロード
- トラブルシューティング
- システム要件

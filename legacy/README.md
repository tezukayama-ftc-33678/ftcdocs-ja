# Legacy フォルダ

このフォルダには、古くなったスクリプトやガイドドキュメントを保管しています。

## 📁 構成

- **`guides/`** - 古いガイドドキュメント
- **`scripts/`** - 古いヘルパースクリプト
- **`root-scripts/`** - ルートディレクトリにあった古いスクリプト

## ⚠️ 注意

これらのファイルは**参考用**として保存されていますが、現在のワークフローでは使用していません。

**現在のワークフロー**については、[../WORKFLOW.md](../WORKFLOW.md) を参照してください。

---

## メインスクリプト（現行版）

現在使用されているメインスクリプトは以下の3つです：

1. **翻訳**: `scripts/translate_po_smart.py`
2. **テスト**: `scripts/test_simplified_chinese_detection.py`
3. **修正**: `scripts/normalize_po_files.py`

これ以外のスクリプトは、特定の目的で作成されたものや、古いワークフロー用のものです。

---

## なぜlegacyに移動したのか？

プロジェクトが成長するにつれて、多くのスクリプトやガイドドキュメントが作成されましたが：

- 重複する内容のガイドが増えた
- 古いワークフローのスクリプトが残っていた
- 後輩たちが「何から始めればいいか」わからなくなった

そのため、**シンプルで明確なワークフロー**を提供するために、必要最小限のスクリプトとドキュメントに絞り込みました。

---

## Legacy ファイルの利用

もし特定の機能が必要な場合は、このフォルダ内のスクリプトを参考にしてください。

ただし、これらのスクリプトは：
- メンテナンスされていない可能性があります
- 現在の依存関係で動作しない可能性があります
- より良い代替手段が現行スクリプトにある可能性があります

**推奨**: まず現行のワークフローで解決できないか確認してから、legacyファイルを参照してください。

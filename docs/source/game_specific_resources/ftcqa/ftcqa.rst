FIRST Tech Challenge ゲームQ&A
==============================

`ゲームQ&A <https://ftc-qa.firstinspires.org/>`_ は、チームが現在のシーズンのチャレンジに関する説明を
ゲームデザイン委員会から受けるための機会を提供するフォーラム/ツールです。Q&Aでの判定は最終的かつ拘束力があり、
競技マニュアルに反映されます。 

質問方法
~~~~~~~~

シーズンの `ゲームQ&A <https://ftc-qa.firstinspires.org/>`_ が開設されると
（通常、チャレンジ発表から2〜3週間以内）、チームは `FIRST Dashboard <https://my.firstinspires.org/Dashboard/>`_
アカウント経由で提供される固有のアカウントを使用して質問できます。チームは
これらの認証情報を使用してシステムにログインし、質問することができます。
誰でも個人アカウントを作成して質問を閲覧またはタグ付けすることはできますが、質問できるのは
チームに提供されたアカウントのみです。

質問を行うチームアカウントの認証情報の取得は、リードコーチ1またはリードコーチ2のみが実行でき、
`FIRST Dashboard <https://my.firstinspires.org/Dashboard/>`_ のチーム情報の「Team Options」列にある
「Payment & Product」ドロップダウンから「Passwords/Voucher Codes」を選択します。
チームのQ&Aウェブサイト認証情報は、結果として表示されるウェブページの「Game Q&A
Forum Accounts」セクションに記載されています。
 
ゲームQ&Aサマリー
~~~~~~~~~~~~~~~~~

ゲームQ&Aツールは、回答されたすべての質問の
「`1ページサマリー <https://ftc-qa.firstinspires.org/onepage.html>`_」を定期的に更新します。
この1ページサマリーには、すべての質問と回答が含まれることは保証されておらず、
定期的に更新されることも保証されていませんが、フォーラム上の質問と回答の
印刷可能な形式を取得する最良の方法です。

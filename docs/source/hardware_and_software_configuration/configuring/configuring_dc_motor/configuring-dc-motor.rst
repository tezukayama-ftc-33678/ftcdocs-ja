DC モーターの構成
=======================

ファイルを作成したので、DC モーターを構成ファイルに追加する必要があります。

.. important:: この時点では、構成ファイルを作成しましたが、その内容を **Robot Controller** にまだ保存していません。構成ファイルは、後の :ref:`構成情報の保存 <hardware_and_software_configuration/configuring/saving_config/saving-config:Saving the Configuration Information>` ステップで保存します。

DC モーターの構成手順
-----------------------------------

1. 画面の **Motors** という単語をタッチして、モーター構成画面を表示します。

.. image:: images/ConfiguringHardwareMotorStep1.jpg
   :align: center

|

2. モーターを **Expansion Hub** のポート #0 に取り付けたので、ポート 0 のドロップダウンコントロールを使用して、モータータイプ（この例では Tetrix Motor）を選択します。

.. image:: images/ConfiguringHardwareMotorStep2.jpg
   :align: center

|

3. タッチスクリーンキーパッドを使用して、モーターの名前を指定します（この例では「motorTest」）。

.. image:: images/ConfiguringHardwareMotorStep3.jpg
   :align: center

|

4. **Done** ボタンを押して、モーター構成を完了します。アプリは前の画面に戻るはずです。

.. image:: images/ConfiguringHardwareMotorStep4.jpg
   :align: center

|

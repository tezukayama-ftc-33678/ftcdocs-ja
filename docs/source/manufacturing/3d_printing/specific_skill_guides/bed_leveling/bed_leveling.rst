ベッドレベリング
============

.. note:: このガイドは主に自動レベリング機能を持たない古いプリンター向けです。新しいプリンターをお持ちの場合、このプロセスは
          必要ない可能性があるか、より高度なプロセスに従う必要があります。プリンターで利用可能なプロセスを調査してください。

ネジで調整可能なベッドをお持ちの場合、ベッドレベリングは、プリントがベッド全体にどれだけよく接着するかの最大の要因です。必要なツールは、
0.2mm厚の金属片、カードストックの一片、またはインデックスカードだけです。

.. image:: images/bedlevelingmaterials.png
  :align: center
  :width: 55%
  :alt: ベッドレベリングに使用可能なツール。

|

ベッドをレベリングするために従うべき手順は次のとおりです：

* ノズルがきれいであり、ベッドレベリングの精度を妨げる余分なフィラメントがその下にないことを確認します。
* Z軸を完全にホームし、プリンターの移動コマンドを使用してZ座標0にします
* プリンターの移動コマンドを使用して、ノズルをネジの1つのすぐ上に移動します（これは、プリンターが独自の制御下で実行する動きを複製することを確認するために移動コマンドで行われます）
* 0.2mm材料がノズルの下でわずかな摩擦でかろうじてスライドできるまで、ネジを上下に調整します。

.. image:: images/bedlevelingheight.png
  :align: center
  :width: 55%
  :alt: ノズルの下にツールを配置。


* ベッドで利用可能な他のネジでこのプロセスを繰り返します。
* ノズルをベッドの中央に移動し、0.2mm材料を使用してベッドの高さを確認します。すべてのネジを適切にレベリングした場合、ほぼ同じ量の摩擦であるはずです。

Zオフセット
--------

プリンターがネジの間でレベリングされると、ベッドは平らになりますが、常に正確な高さにあるとは限りません。すべてのプリンターには、
プリント中またはプリント前にノズルの高さを調整できるZオフセット変数があります。プリントの開始時にベッドが高すぎるか低すぎる場合でも、一貫している場合
（ベッドレベリングを提供した場合）、プリンターのZオフセット機能（ベビーステッピングとも呼ばれる）を使用して、最初の層を配置している間にライブで調整します。適切な高さにあることを確認するための
いくつかの良い視覚化については、以下の画像を参照してください。

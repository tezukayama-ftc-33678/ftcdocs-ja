まとめ: myBlocksのメリット
=============================

1. MyBlocksは、Software Development Kit (SDK) のJavaの全範囲へのアクセスを提供するようになりました。
   Blocksプログラミングは、**以前はBlocks専用チームには利用できなかった** タスクを実行できるようになりました。
   これには :ref:`外部ライブラリ <programming_resources/shared/external_libraries_blocks/external-libraries-blocks:external 
   libraries in onbot java and blocks>` も含まれます。

2. MyBlocksは、以前は **長いまたは複雑な関数** をBlocksできちんとパッケージ化できます。

3. MyBlocksプログラミングにより、チームの一部のメンバーはJavaの学習と使用を開始し、
   貴重な新機能を提供できます。他のチームメンバーはBlocksで学習と作業を続け、
   チームの公式コードを作成できます。誰も取り残されることはありません。

4. MyBlocksは **OnBot Java** で作成できます。これはRC電話またはControl Hubで実行されます。
   ビルドとテストは非常に高速です。多くのチームは、学校のコンピュータがソフトウェアの
   インストールを防ぐなどの理由で、Android Studioに簡単にアクセスできません。

5. myBlocksを開発して共有することで、経験豊富なチームは、単にJavaライブラリへのリンクを
   投稿するだけでなく、**新しいチームをより直接的に支援** できます。*FIRST* Tech Challenge
   コミュニティは、最終的には、テストされ、十分に文書化されたmyBlocksのキュレートされた
   リポジトリの恩恵を受ける可能性があります。おそらく「Blocks Store」でしょうか？

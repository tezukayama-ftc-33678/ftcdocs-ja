   **TFOD** はこの認識を行いましたか？


これはどうしてでしょうか？答え：この画像は「instant」の結果ではありませんでした。**TFOD** が Duck を認識した** 後** 、露出が非常に低く減らされました。

**TensorFlow Lite** （および **Vuforia** ）の実装は、現在識別されているオブジェクト（または画像）を、平行移動、回転、部分的なブロッキング、さらには露出の極端な変化を通じて** 追跡** するのが得意です。

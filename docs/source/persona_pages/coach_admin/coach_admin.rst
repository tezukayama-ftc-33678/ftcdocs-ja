コーチ（管理）リソース
================================

*FIRST* Tech Challenge コーチのページへようこそ！リソースは種類別に整理されており、チームがシーズンを通して組織的かつ円滑に活動できるようになっています。これらのリソースは、*FIRST* の理念を推進しながらチームを管理するコーチや管理者のニーズに焦点を当てています。探したいリソースのボタンをクリックするだけです！
